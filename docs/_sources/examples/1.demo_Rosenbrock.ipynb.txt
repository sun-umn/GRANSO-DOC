    "# Rosenbrock\n",
    "\n",
    "Minimize 2-variable nonsmooth Rosenbrock function, subject to a simple bound constraint taken from: Curtis, Frank E., Tim Mitchell, and Michael L. Overton. \"A BFGS-SQP method for nonsmooth, nonconvex, constrained optimization and its evaluation using relative minimization profiles.\" Optimization Methods and Software 32.1 (2017): 148-181."
   ]
  },
  {
   "cell_type": "markdown",
   "id": "eaeebfa4",
   "metadata": {},
   "source": [
    "## Problem Description"
   ]
  },
  {
   "cell_type": "markdown",
   "id": "7a94952c",
   "metadata": {},
   "source": [
    "$$\\min_{x_1,x_2} w|x_1^2-x_2|+(1-x_1)^2,$$\n",
    "$$\\text{s.t. }c_1(x_1,x_2) = \\sqrt{2}x_1-1 \\leq 0, c_(x_1,x_2)=2x_2-1\\leq0,$$\n",
    "\n",
    "where $w$ is a constant (e.g., $w=8$)"
   ]

    "## Problem Definition \n",
    "\n",
    "Specify torch device, optimization variables, and objective and constraint(s).\n",
    "\n",
    "Note: please strictly follow the format of comb_fn, which will be used in the NCVX main algortihm."

  {
   "cell_type": "code",

   "source": [
    "device = torch.device('cpu')\n",
    "# variables and corresponding dimensions.\n",
    "var_in = {\"x1\": [1,1], \"x2\": [1,1]}\n",
    "\n",
    "def comb_fn(X_struct):\n",
    "    x1 = X_struct.x1\n",
    "    x2 = X_struct.x2\n",
    "    # enable autodifferentiation\n",
    "    x1.requires_grad_(True)\n",
    "    x2.requires_grad_(True)\n",
    "    \n",
    "    # objective function\n",
    "    f = (8 * abs(x1**2 - x2) + (1 - x1)**2)\n",
    "\n",
    "    # inequality constraint, matrix form\n",
    "    ci = GeneralStruct()\n",
    "    ci.c1 = (2**0.5)*x1-1  \n",
    "    ci.c2 = 2*x2-1 \n",
    "\n",
    "    # equality constraint \n",
    "    ce = None\n",
    "\n",
    "    return [f,ci,ce]"
   ]
