Home
=======================================



.. warning::
   The software is under development. Use it at your own risk, and please send your feedback to Buyun Liang via liang664@umn.edu.


NCVX is a python numerical package revamped and translated from the original GRANSO package. 
In order to lower the barriers to general users and solve modern machine/deep learning problem, 
we introduced several main features including auto-differentiation, GPU acceleration, tensor input, 
scalable QP solver and improved core algorithms.

Get the Code
-------------

The source code is available on Github (https://github.com/sun-umn/NCVX).
