Citing NCVX
============

    title={NCVX: A User-Friendly and Scalable Package for Nonconvex Optimization in Machine Learning},
    author={Liang, Buyun and Sun, Ju},
    journal={arXiv preprint arXiv:2111.13984},
    year={2021}
  }
